# ProjetoAlunos


O programa mostra o seguinte Menu:

1 : Inserir Alunos
2 : Mostrar Alunos
3 : Apagar Todos os Dados
4 : Mostrar Sala 

5 : Sair 

Descrição----------------------------------------

1 - Insere dados dos alunos
2 - Mostra os alunos cadastrados
3 - Apaga o arquivo criado em 1
4 - Mostra os alunos contidos na sala selecionada.
